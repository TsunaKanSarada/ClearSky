# CrearSky - React + Vite + Go(仮)

偏頭痛とかを教えてくれます

## 注意
> [!WARNING]
> - WSLは使用しないでください
> - firebase cliへのlogin忘れずに

## Setup

### インストール
#### Mac
miseを入れたらあとはよしなにやってくれます
- [mise](https://mise.jdx.dev/getting-started.html)

#### windows
手動デス
- **必須**
  - [Taskfile](https://taskfile.dev/installation/)
  - [Bun](https://bun.sh/)
- 任意
  - [Go](https://go.dev/)

### 実行

#### Mac
```shell
mise install
task pre
```

#### Windows
```shell
task pre
```

## Dev
```shell
task dev
```
